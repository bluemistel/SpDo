# SpDo
推しの音声を通知サウンドに設定できるタイマー付きのミニマルなデスクトップToDoアプリです。

## 特徴

- **カンバン & リスト表示**: タスクを視覚的なカンバン形式、またはリスト形式で整理できます。
- **ポモドーロタイマー**: カウントダウン、ストップウォッチ、ポモドーロタイマーを切り替え可能。
- **カスタマイズ可能なテーマ**: ダークモード対応の7種類の鮮やかなカラーテーマ。
- **サウンド通知**: タスク完了やタイマーイベントに合わせて、お気に入りのサウンドで通知を設定できます。
- **自動保存**: タスクや設定はローカルに自動的に保存されます。

## 開発方法

```bash
# 依存関係のインストール
npm install

# 開発モードで実行
npm run dev

# 本番用にビルド
npm run build
```

## GitHub リリースの作成手順

1.  **コミットとプッシュ**:
    ```bash
    git add .
    git commit -m "Initial release"
    git push origin main
    ```
2.  **タグの作成**:
    ```bash
    git tag v1.0.0
    git push origin v1.0.0
    ```
3.  **ビルド**:
    `npm run build` を実行して、`out` ディレクトリに本番用ファイルを生成します。
4.  **公開**:
    GitHubのリポジトリ `https://github.com/bluemistel/SpDo` (未作成の場合は作成) にアクセスし、「Releases -> Create a new release」からビルド成果物をアップロードして公開します。
